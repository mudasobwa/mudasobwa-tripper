mudasobwa-tripper

Web component reads the images list (with optional header and intro) from JSON file:

      {
        "header": "Sant Feliu de Guíxols",
        "intro": "Sant Feliu de Guíxols (Catalan pronunciation: [ˈsam fəˈɫiw ðə ˈɣiʃuɫs ...",
        "images": [
          "i/9c8e7dcc4c084dcaf0f33d38d9d51a02-600.jpg",
          "i/dc3027c3d68d44691aaebe3f874812da-600.jpg",
          "i/6a205e45b266c6f812f8a8ff61b17e58-600.jpg"
        ]
      }

...and displays everything on the page, using mudasobwa-exthen component.

    <mudasobwa-tripper src="data/images.json">
    </mudasobwa-tripper>
